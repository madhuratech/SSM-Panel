# Frontend Application

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=http://localhost:5000/api
```

3. Start development server:
```bash
npm run dev
```

App runs on: http://localhost:3001

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
